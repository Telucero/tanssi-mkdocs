    Executar um Nó¶

    A configuração de um nó de rede é essencial para interagir com a Tanssi ou sua rede Powered by Tanssi, fornecendo um endpoint RPC seguro e dedicado. Esta seção oferece guias passo a passo para ajudá-lo a configurar e gerenciar diferentes tipos de nós de forma eficaz.

    • Nó Tanssi - um nó Tanssi se conecta diretamente à rede Tanssi, permitindo que você interaja via endpoint RPC
    • Nó de Rede Powered by Tanssi - execute um nó para sua rede Powered by Tanssi, permitindo que você mantenha seu próprio endpoint RPC

    Explore Esta Seção¶

    Nó Tanssi

    Aprenda como configurar e executar um nó Tanssi usando Docker ou Systemd para interagir com a rede Tanssi.

    Nó de Rede com Tanssi

    Aprenda como configurar e executar um nó em uma rede Tanssi-powered usando Docker ou Systemd, o que permite que você hospede seu próprio endpoint RPC para interação com a cadeia.
